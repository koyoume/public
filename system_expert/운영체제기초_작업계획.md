# 운영체제 기초 — 작업 계획 및 변경 이력

## 📋 작업 개요

서울대학교 홍성수 교수님 「운영체제의 기초」 강의 전체 12개 챕터를 정리한 학습 자료.

### 사용자 요구사항 (누적)

1. **계획 먼저 논의 후 작업 시작**
2. **변경 사항을 주기적으로 md로 정리**
3. **구체적 요구 사항을 md로 누적 업데이트**

### 작성 형식 (챕터 정리 워크플로우)

각 챕터마다:
- **PDF → ZIP 변환 후 슬라이드 추출**
- **모든 슬라이드 읽기 (텍스트 + 이미지)**
- **한국어로 정리**: 동기 → 직관 → 원리 → ASCII 다이어그램 → 코드/공식 보존 → 흔한 오해
- **trigger 명령**:
  - `"다음"` → 다음 챕터로
  - `"다음 반영"` → GitHub에 반영

### 출력 파일 형식 (3종)

1. **Knowledge MD**: 시험 최적화 (수식표, 비교표, 용어정의, 상세토픽, exam checklist)
2. **Knowledge PDF**: pandoc 변환 (특정 헤더 제거: 첫 H1, 업데이트 정보 라인, ##①~④ 서브헤더, ⑤ exam checklist)
3. **수업정리노트 PDF**: 전체 강의 노트 형식

### Pandoc 변환 명령

```bash
# Knowledge PDF (margin 15mm)
pandoc [md] -o [pdf] --pdf-engine=wkhtmltopdf -V geometry:margin=15mm -V papersize:a4

# 수업정리노트 PDF (margin 20mm)  
pandoc [md] -o [pdf] --pdf-engine=wkhtmltopdf -V geometry:margin=20mm -V papersize:a4
```

## 📚 완료된 챕터 (12/12)

| # | 챕터 | 핵심 주제 |
|---|------|----------|
| 01 | Intro | OS 진화 단계 (Phase I/II/III), Coordinator/Illusion Generator/Standard Library, base/bound MMU |
| 02 | Hardware | von Neumann, FDE cycle, system bus, interrupt, hardware protection (dual mode, I/O, memory, CPU) |
| 03 | Runtime Stack | x86 calling convention (cdecl), stack frame, prologue/epilogue, main+add 추적 |
| 04 | Process and Threads | PCB, state transition, context switching, fork/exec/wait, COW, ULT/KLT/Combined |
| 05 | CPU Scheduling | FIFO, SJF/SRTF, RR, MLFQ, Fair Share (GPS/WRR/WFQ), Linux CFS (RB-tree, nice 10% rule) |
| 06 | Synchronization | Race condition, semaphore (P/V), producer-consumer, condition variable, monitor |
| 07 | Deadlocks | 4 조건 (Coffman), prevention, Banker's algorithm, detection/recovery |
| 08 | GNU Linker | ELF, section, symbol resolution (strong/weak), relocation, static/dynamic linking |
| 09 | Dynamic Memory Allocation | Fragmentation, implicit/explicit/segregated list, GC (mark-sweep, ref counter) |
| 10 | Segmentation and Paging | Multiple segments, page table, paged segmentation, TLB (EAT, MIPS R2000/R3000, ASID) |
| 11 | Demand Paging | Virtual memory, page fault, replacement (OPT/LRU/Clock/FIFO), working set, thrashing, 4-level paging |
| 12 | Device Drivers | I/O controller, character/block/network devices, disk/flash, cdevsw/bdevsw, 2-level interrupt |
