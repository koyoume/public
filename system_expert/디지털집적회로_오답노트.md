## Q6 — Ch2 단채널 효과 | **부분 정답**

**문제**: 채널 길이 L을 절반으로 줄였을 때 DIBL, 속도 포화, 서브스레숄드 스윙 S의 변화

**내 답변**: DIBL: Vth 감소. 속도 포화: Ids가 (Vgs-Vt)에 선형 비례. S 변화: 미답

**정답 및 보완**:

1. **DIBL**: L 감소 → Drain 공핍층이 Source 장벽에 영향 → Vth 감소
2. **속도 포화**: 단채널에서 전기장 강해짐 → vsat 도달 → Ids가 (Vgs-Vt)^2에서 (Vgs-Vt)로 선형화.
   Vgs 대비 전류 효율 저하 (최대 Ids 절대값 감소가 아닌 증가율 둔화)
3. **S 변화**: DIBL로 n 증가 → S = (kT/q)ln(10)*n 증가 → 누설 전류 제어 어려워짐

**핵심 개념**: S = (kT/q) * ln(10) * n >= 60mV/dec. 채널 단축시 n 증가로 S 악화

## Q7 — Ch4 Logical Effort Path 계산 | **정답**

**문제**: NAND2 → NOR2 → INV 경로에서 PE, N, f-hat, 각 단 Cin 계산 (Wi → 27Wi)

**내 답변**: PE = 59.94, N = 3단, f-hat = 3.86. Cin_inv=6.99, Cin_nor=3.02, Cin_nand=1.04

**정답 및 보완**:

- G = 1 x 4/3 x 5/3 x 1 = 20/9 = 2.22
- EF = 27/1 = 27, B = 1
- PE = 2.22 x 27 = 59.94
- N = log4(59.94) = 2.95 → 3단
- f-hat = 59.94^(1/3) = 3.86
- 뒤에서 앞으로: Cin_inv = 27/3.86 = 6.99, Cin_nor = (6.99 x 5/3)/3.86 = 3.02, Cin_nand = (3.02 x 4/3)/3.86 = 1.04 ≈ Wi (입력과 일치 확인)

**핵심 개념**: 뒤에서 앞으로 Cin 계산: Cin_i = (Cout_i x LE_i) / f-hat. 최종값이 Wi와 일치하는지 검증 필수

## Q8 — Ch6 Pulsed Latch Time Borrowing | **부분 정답**

**문제**: Tc=10ns, tsetup=0.5ns, Tpw=2ns. 최대 tpd 증가량, Hold 조건 강화, Tpw 무한대 불가 이유

**내 답변**: 1. 2ns. 2. tcd >= thold - tccq + Tpw. 3. Hold time 위반 위험만 언급

**정답 및 보완**:

1. Tpw = 2ns만큼 tpd 허용량 증가 ✓
2. tcd >= thold - tccq + Tpw ✓
3. **Tpw 무한대 불가 이유**:
   - Hold 조건 극도로 강화 → tcd를 무한히 크게 할 수 없음
   - 투명 구간이 길어지면 더블 샘플링 문제 재발
   - tborrow 물리적 상한: tborrow_max = Tc/2 - tsetup = 10/2 - 0.5 = 4.5ns

**핵심 개념**: Tpw가 클수록 느린 경로에 유리하지만 Hold 위반 위험 증가. 상한 = Tc/2 - tsetup

## Q9 — Ch7 다이나믹 회로 전하 공유 | **정답**

**문제**: C2=100fF(VDD), C1=50fF(0V). 전하 공유 후 출력 전압 계산 및 해결책

**내 답변**: V(출력) = 100/(150) * VDD = 1.2V. 해결책: C1도 프리차지

**정답 및 보완**:

- V(X2) = VDD * C2/(C1+C2) = 1.8 * 100/150 = 1.2V
- 전압 강하 = 0.6V. 인버터 트립 전압(≈0.9V) 이상이므로 이 경우는 오동작 없음
- 단, C1이 더 크거나 VDD가 낮으면 트립 전압 아래로 떨어져 오동작 가능
- **해결책**: 프리차지 구간(CLK=0)에 C1 노드에도 약한 PMOS 연결 → VDD로 충전 → 평가시 전압 변화 없음

**핵심 개념**: V(X2) = VDD * C2/(C1+C2). 내부 노드 프리차지로 전하 공유 방지

## Q10 — Ch8 DRAM vs eDRAM Gain Cell | **정답**

**문제**: 조건A(CPU 다이, 읽기 빈번) vs 조건B(독립칩, 최대 용량) 메모리 선택

**내 답변**: A: eDRAM Gain Cell (비파괴 읽기). B: DRAM (면적 유리)

**정답 및 보완**:

**조건 A → eDRAM Gain Cell**:
- 읽기/쓰기 경로 분리 → 비파괴 읽기 → 복원 불필요 → 읽기 지연 감소
- PMOS 게이트 누설 < NMOS 기판 누설 → 리프레시 주기 연장
- CPU와 같은 다이 → 통신 지연 최소화

**조건 B → DRAM(1T-1C)**:
- 셀당 1T+1C → 최소 면적 → 최대 집적도
- Gain Cell(3T)은 동일 면적에 1/3 용량

**핵심 개념**: Gain Cell 장점: 비파괴 읽기, 낮은 누설. DRAM 장점: 최소 면적(1T-1C)

## Q11 — Ch3+Ch6 전력 vs 지연 트레이드오프 | **부분 정답**

**문제**: VDD=1.0V→0.7V 감소 시 동적 전력 감소율 및 Setup Time 위반 여부

**내 답변**: 1. 0.1mV (단위 오류). 2. 51% 감소. 3. tpd=0.604ns → 위반 없음 (계산 오류)

**정답 및 보완**:

1. P = CL x VDD^2 x f = 100fF x 1^2 x 1GHz = 100uW (단위: W)
2. P_new/P_old = (0.7/1.0)^2 = 0.49 → 51% 감소 ✓
3. tpd_new = 0.4 x (1.0/0.7)^2 = 0.4 x 2.04 = 0.816ns
   Tc >= tpcq + tpd + tsetup = 0.1 + 0.816 + 0.1 = 1.016ns > 1ns → **Setup Time 위반 발생!**
   (1/0.7)^2 = 2.04이므로 1.51이 아님

**핵심 개념**: VDD 감소시 전력은 제곱 감소, 속도도 제곱으로 느려짐. 동시 최적화 필요

## Q12 — Ch4+Ch1 복합 게이트 Cin/LE 계산 | **모름**

**문제**: Y = NOT((A AND B) OR C) 단일 복합 게이트의 PDN/PUN, Cin(A), Cin(C), LE(A)

**정답 및 보완**:

PDN: (A,B 직렬 NMOS) 와 (C NMOS) 병렬
PUN: (A,B 병렬 PMOS) 와 (C PMOS) 직렬

Cin(A): PMOS_A(병렬→2W) + NMOS_A(직렬→2W) = 4Cg → LE_A = 4/3 (NAND2와 동일)
Cin(C): PMOS_C(직렬→4W) + NMOS_C(병렬→1W) = 5Cg → LE_C = 5/3 (NOR2와 동일)

**핵심 개념**: 복합 게이트에서 각 입력의 LE는 해당 입력이 PDN/PUN에서 직렬/병렬 위치에 따라 결정됨

## Q13 — Ch5 쉴드 삽입 트레이드오프 | **정답**

**문제**: 동상/역상 스위칭에서 쉴드 삽입 전후 Ceff 변화 및 단점

**내 답변**: 동상: 쉴드 전후 Cself로 동일. 역상: 2Cc 제거 효과. 단점: 면적 증가

**정답 및 보완**:

- 동상 스위칭: Ceff = Cself (전후 동일, 쉴드 효과 없음)
- 역상 스위칭: 쉴드 전 Cself+2Cc → 쉴드 후 Cself (2Cc 제거, 효과 매우 큼)
- 추가 단점: 배선 채널 소비 증가, 라우팅 복잡도 증가

**핵심 개념**: 쉴드는 역상 스위칭일 때 효과 극대화. 동상은 이미 상쇄되므로 불필요

## Q14 — Ch8 DRAM 읽기 전압 계산 | **부분 정답**

**문제**: Ccell=25fF, Cbl=500fF, VDD=1.5V. 1저장시/0저장시 dVbl, 행 2배시 변화

**내 답변**: 1. 35mV ✓. 2. 0.17mV (오답). 3. 절반으로 감소 ✓

**정답 및 보완**:

1. dVbl("1") = 25/525 x 0.75 = 35.7mV ✓
2. dVbl("0") = -(25/525) x 0.75 = -35.7mV
   "0"은 Vcell=0V이므로 BL에서 셀로 전하가 이동 → 음의 방향
3. Cbl 2배(1000fF): dVbl = 25/1025 x 0.75 = 18.3mV ≈ 절반 ✓

**핵심 개념**: "0" 저장시 dVbl = -(Ccell/(Ccell+Cbl)) x (VDD/2). 부호가 반대

## Q15 — Ch7+Ch9 다이나믹 회로와 SRAM 읽기 경로 | **모름**

**문제**: SRAM 읽기 경로에 다이나믹 회로 사용 이유 (와이드 OR, 단조성, Footed/Unfooted)

**정답 및 보완**:

1. 와이드 OR 구조: PMOS PUN 없이 NMOS PDN만으로 수백 셀 병렬 연결 가능.
   프리차지 후 하나만 켜져도 즉시 방전 시작 → 빠른 읽기
2. 단조성: BL은 프리차지(VDD)→방전(GND) 방향만 가능.
   다이나믹 회로 출력도 1→0만 가능 → 물리적으로 일치하여 자연 보장
3. Footed 사용 필요: BL은 프리차지로 VDD(=1) 상태에서 평가 시작.
   Unfooted는 입력=1인 상태에서 프리차지 PMOS와 충돌 발생 -> Footed 필수.
   FF 직후 첫 단과 동일한 이유.

**핵심 개념**: 다이나믹 회로의 단조성이 BL 방전 특성과 일치 → SRAM 읽기에 최적
